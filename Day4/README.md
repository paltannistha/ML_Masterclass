# 📊 Pass/Fail Prediction (Classification)

A simple Machine Learning project using **Logistic Regression** to predict whether a student will pass or fail based on study hours.

---

## 🚀 Objective
- Understand **classification**
- Build a basic ML model
- Evaluate using **accuracy**

---

## 📂 Dataset
- Input: Hours studied  
- Output: `0 = Fail`, `1 = Pass`

---

## ⚙️ Tech Stack
- Python
- NumPy
- scikit-learn

---

## 🛠️ Steps
1. Create dataset  
2. Train model  
3. Make predictions  
4. Calculate accuracy
