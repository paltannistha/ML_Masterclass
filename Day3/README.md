# Linear Regression Example

This project demonstrates a simple **Linear Regression model** using Python and Scikit-learn.  
The model predicts **house price based on house size**.

## Dataset

| Size (sq ft) | Price (thousands) |
|--------------|-------------------|
| 500 | 50 |
| 800 | 80 |
| 1000 | 100 |
| 1200 | 120 |
| 1500 | 150 |

## Technologies
- Python
- NumPy
- Matplotlib
- Scikit-learn

## How it Works
1. Create a dataset of house sizes and prices.
2. Train a Linear Regression model.
3. Predict the price for a new house.
4. Visualize the regression line.
