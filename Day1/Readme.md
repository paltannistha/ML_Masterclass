# Day 1: Introduction to Machine Learning

## What is Machine Learning?

Machine Learning is a field of **Artificial Intelligence** where computers learn patterns from data and make predictions without being explicitly programmed.

In simple words:

> Machine Learning means teaching computers to learn from data.

Instead of writing rules manually, we give the machine data and it learns the rules by itself.

---

## Real Life Examples

Machine Learning is used in many applications:

- Netflix recommending movies
- Gmail detecting spam emails
- Google Maps predicting traffic
- Instagram suggesting posts

---

## Types of Machine Learning

There are three main types of Machine Learning.

### 1. Supervised Learning

In supervised learning, the data has **labels**.

Example:

| Hours Studied | Marks |
|---------------|------|
| 2 | 40 |
| 4 | 60 |
| 6 | 80 |

The model learns the relationship between hours studied and marks.

It can then predict marks for new data.

Example applications:
- House price prediction
- Spam detection

---

### 2. Unsupervised Learning

In unsupervised learning, the data **does not have labels**.

The algorithm tries to find patterns or groups in the data.

Example:

| Customer | Age | Spending |
|----------|-----|---------|
| A | 22 | High |
| B | 45 | Low |
| C | 23 | High |
| D | 50 | Low |

The machine groups similar customers together.

This process is called **Clustering**.

---

### 3. Reinforcement Learning

In reinforcement learning, the machine learns through **rewards and penalties**.

Example:
- Training robots
- Self-driving cars
- Game playing AI

The system improves by learning from its actions.

---

## Basic Machine Learning Workflow

1. Collect Data  
2. Prepare and clean the data  
3. Train the machine learning model  
4. Test the model  
5. Make predictions

---

## Key Idea

Traditional Programming:
Rules + Data → Output

Machine Learning:
Data + Output → Learn Rules
